Mental Health Analysis – Mid Project

By: Karim Deheya

Overview

This project explores mental-health patterns among employees based on gender, age, company size, work environment, and country.
The analysis was done using survey data to understand who is most affected, who seeks treatment, and how workplace factors influence mental health.
